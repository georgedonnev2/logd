初次开机

烧录系统镜像

选取了 OrangePi AIpro 对应 Ubuntu 镜像。选取了其中的 opiaipro_ubuntu22.04_desktop_aarch64_20241128.img.xz 镜像，可以有桌面，5GB多，从百度网盘下载。

烧录过程略。

连接外设

插上 HDMI 显示器。需要注意的是，官网说明要插鲲鹏开发板的 HDMI0，对应于鲲鹏开发板金属外壳标注的 HDMI-2。金属外壳上盖，可以取下（不需要拧螺丝），然后看到开发板上的HDMI0，对应外壳的HDMI-2。

连接USB鼠标、USB键盘等，此处从略。

插上电源

运行预置样例

系统启动完毕后，Desktop 显示 HwHiAiUser 登录。输入密码 Mind@123

参考 昇腾开发板运行预置样例，运行预置样例。

ssd样例。未运行成功，报错：缺少mindspore xxx。（可能操作有误，或其他原因，未进一步排查）

ResNet样例。运行成功。结果有：猫、船、飞机，等。

初步判断：鲲鹏开发板（OrangePi Kunpeng Pro），可以当做昇腾开发板使用。

修改 IP 地址

修改 IP 地址为固定 IP 地址，以便后续用 ssh 访问。

需要注意的是，通过界面修改 IP 地址时，链接eth1其实是不存在（没有对应物理的 device。估计是没有改全），因此怎么修改都不成功（也不报错）。实际上只有 1 个网口，对应的物理 device是 eth0。要么修改链接 eth1 对应的 device 为 eth0；或者新建一个链接，选择对应的物理 device 是 eth0。

链接名称和真正的物理 device 对应上了，修改 IP 地址就可以生效了。修改 IP 地址的过程，此处从略。
