职位13： 电子电器开发工程师
地点：无锡

职位亮点：
1. 年轻、多元化的团队，具有包容、团结和积极乐观的工作氛围。
2. 国际化的研发团队，大量学习及培训机会，多元化的职业晋升通道，打造个人的职业成功。
3. 您将加入一个具有创新精神的精英团队，同时还可以从触手可及的庞大经验资源网络中受益。
4. 作为电子电器零部件工程师，您将和卡特全球团队合作开发零部件用于卡特重载机器，以确保产品具有卓越的性能、耐用性、质量和可维护性。
5. 电子电气零部件售后组，负责采用大数据分析、机器学习、AIGC等先进技术及方法，为客户提供数据增值服务，帮助代理商实现客户需求，以促进电子电气零部件或系统在中国市场的销售与推广，实现售后业务的持续增长。

职位描述：
1. 根据电子电气零部件或液压原件工作原理，结合客户设备的使用环境和操作习惯，利用后台已记录数据，找出数据特征。利用机器学习算法，进行数据分析，实现零部件故障分析、预测和预测剩余使用寿命，
2. 负责基本数据清洗，优化算法模型，提高分析精度和预测准确度，创建可视化报表
3. 收集客户需求转发为产品技术要求，按时完成项目开发工作
4. 严格按照项目开发流程进行开发设计，并进行相关文档编写整理工作，按项目流程完成阶段性的项目评审工作
5. 负责公司电子电器零部件的软件开发工作，为各种电子控制模块开发C, C#和/c++或Python嵌入式软件

技能和经验要求：
1. 全日制大学本科及以上，物联网工程、计算机科学、人工智能、数据分析、电子信息、自动化、电气工程等理工科相关专业优先。
2. 熟悉嵌入式电子产品原理图设计或者PCB设计，能使用一种或以上原理图或者PCB绘图工具。
3. 至少熟练掌握一种Python或C/C++等编程语言，了解MATLAB/Simulink仿真。
4. 熟练掌握机器学习基本算法，能根据相关业务逻辑建立机器学习数学模型，有机器学习建模、算法优化等项目经历，或机械工程制造企业实习经历优先。
5. 工作认真负责积极主动，主观能动性强，善于思考，具有上进心，具有较强的数据敏感度。
6. 思路严谨，具有良好的心理素质及团队沟通、合作能力，良好的英语表达能力。

职位14：嵌入式软件工程师
地点：无锡

职位概要：
作为 Caterpillar 的嵌入式软件工程师 - 车载远程信息处理系统，您将为支持 Caterpillar 连接程序的车载远程信息处理系统开发嵌入式软件。职责范围包括车载远程信息处理系统的嵌入式软件的配置、集成、构建和测试；嵌入式远程信息处理系统的软件测试包括在工作台环境中开发和验证软件需求以及排除软件或系统缺陷故障。我们拥有一支出色的团队，他们在 Caterpillar 从事一些最令人兴奋的工作，我们迫不及待地希望您加入团队！！
您将做什么：
• 软件变更需求审查和分析。
• 软件集成和配置
• 测试计划创建、执行和文档编制
• 测试台硬件、线束、模型和软件的设置和配置
• 使用手动、自动和模拟方法完成强制性、特定功能和完整回归测试。
• 在实验室、工厂和现场为系统和软件故障排除提供客户支持。

教育要求：
• 需要机电一体化工程、电气工程、计算机工程、计算机科学或机械工程学位。

您需要具备的技能：
技术卓越：
• 深入了解车载远程信息处理产品和应用功能。
• 通过良好的技术能力，有效地检查事件和问题的影响，为日常功能挑战提供有效的技术解决方案。
• 有效地执行技术工作方面，不断积累知识并保持最新状态
• 基于交付产品、技术和服务的卓越表现，应用技术操作和项目标准。
• 应用当前程序和技术来帮助解决个人技术能力一般领域的技术问题。
• 帮助他人解决技术或程序问题。

软件开发：
• 利用专有工具和流程集成和配置远程信息处理 flash 文件。
• 使用 Python 或专有工具开发脚本以自动化测试。

需求分析：
• 遵循确定功能和信息需求的政策、实践和标准。
• 确认与需求分析相关的可交付成果。
• 与客户和用户沟通，以引出并收集客户需求。
• 参与准备详细的文档和需求。

系统测试
• 利用硬件和软件在环等工具和方法测试电子远程信息处理系统（即 dSPACE）的嵌入式软件。
• 应用远程信息处理、传感器、阀门、执行器和 J1939 数据链路通信的电子系统知识，确保嵌入式软件符合法规要求。
• 使用 CANalyzer、CANoe、Caterpillar Electronic Technician 或 Lauterbach 调试器等工具来定义有意义的数据模式。
• 支持项目负责人制定和执行系统测试计划。
• 执行测试用例、分析测试结果并定期报告结果。
• 参与测试系统从硬件或软件故障中恢复的能力。

技术故障排除：
• 发现、分析和解决在测试台、实验室、试验场、现场和工厂发现的硬件、软件或应用程序问题。
• 使用专有和第三方诊断工具。
• 作为高级故障排除的一部分，分析代码、日志和当前系统。
• 记录和报告特定技术问题，解决已使用的流程和工具。

附加信息：
此职位需要的旅行时间不到 10%。
