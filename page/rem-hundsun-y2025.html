恒生电子简介

恒生电子是一家以“让金融变简单”为使命的金融科技公司，总部位于中国杭州。1995年成立，2003年在上海证券交易所主板上市（600570.SH）。

恒生聚焦金融行业，致力于为证券、期货、基金、信托、保险、银行、交易所、私募等机构提供整体解决方案和服务。恒生已连续16年入选FinTech100全球金融科技百强榜单，2023年排名第22位。目前拥有超过13000名员工，其中产品技术人员占比约65%。多年来，恒生以技术服务为核心，凭借多年金融IT建设经验，以及对金融业务的深刻洞察和理解，用优质的产品与服务，持续赋能金融机构创新发展。

岗位分析和建议

岗位官网链接：恒生2025秋季校园招聘。

岗位比较齐全，学院的同学大多符合专业技能和软能力的要求。各位同学可按兴趣和专长，勇于尝试投递相关岗位。能拿到 offer 是很理想的；即使拿不到 offer，也能增长找工作经验。反正没什么损失，至多花了些时间而已。有如下一些分析，供参考。

  • 大多需要熟悉数据库。如果同学有点忘了，赶紧再温习下。
  • 软能力要求也基本都一样：学习能力突出，基础扎实，有责任心，思维活跃，有较好的沟通能力、抗压能和团队协作精神。
  • Java 和 C/C++。甚至还招不熟悉的（但立志从事 Java 和 C/C++），只要有亮眼的作品证明自己超强的学习能力，也是有可能获得 offer的。
  • 交付、维护、运维。3 个岗位的能力要求是一样的，都是技术支持类。应该是工作职责不同，可能是侧重于项目不同阶段。比如维护是项目已经上线运行了，要处理客户遇到的问题等。技术支持类相关，可看 B站视频：恒生情报站—没想到，你是这样的技术支持。
  • 对测试、前端、需求分析岗位感兴趣的，但需要协助的，可联系我。比如，岗位是做什么的；大部分能力具备了，还有1-2个能力需要补，怎么快速补；……
