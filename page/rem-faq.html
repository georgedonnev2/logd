选岗位

最理想最擅长 C 和 C++，其他的语言岗位有必要去试着投嘛

Q--2024/08/28

现在想法就是找开发岗，我最理想也最擅长肯定是 C 和 C++，其它的语言岗位我有必要去试着投嘛，因为我感觉现在学可能多少有点赶。我投过前端开发的，我其实这个方面也感兴趣，但毕竟接触的少，投了就没啥后续了

S--2024/08/29

以下是建议，仅供参考。预祝同学早日找到心仪岗位！

  1. 聚焦投 C 和 C++ 岗位

    • 擅长 + 理想，因此建议聚焦。
    • C / C++ 岗位的薪酬也不错，如能录用，还喜欢工作内容，就很理想！
    • 专业技能（C / C++），以及软能力（沟通 / 表达 / 逻辑 / ……），都符合要求，才能拿到 offer。各方面都不能偏废。
    • 即使满足岗位诸方面要求，也不一定能拿到 offer。可能有更厉害的同学也投递了，尤其是其他 985 和 211 学校的。因此，要能抗压，要尽早动手，则机会更多！
    • 即使很顺利很厉害，手握 10 个 offer，最后也只能去一个单位。因此，心态要平！这么多岗位，我只要找 1 个，肯定是能找到的。
    • 瞄准可能会被录用的企业的岗位。尽量不要在太热门的太高薪的岗位上花太多时间，都知道好，很多牛娃也投，所以和咱们普通人就关系不大。但可能情况下还是要踮脚跳一跳，力争找一个自己能力所匹配薪酬的 1.2 倍甚至 1.5 倍的岗位。
    • 多投、多总结复盘。看看还有哪些不足的，赶紧补上。补的目标是通过面试找到工作，不要无限制的补，没那么多时间。原先只能找 10000 元岗位的，通过不断投岗位、不断获取面试找工作经验、不断补齐能力，最终可能找到 15000 甚至 20000 元的岗位。

  2. 不放弃前端的学习

    • 24 届校友L，通过在 23 年下半年自学增强 Java 技能，花费2~3个月，最终入职了国内著名人工智能上市公司，做 Java 开发。想说的是：如果要学，是来得及的。
    • 学到什么程度？能通过面试找到工作就行。不要无限制的学习，没有那么多时间。
    • 如果有兴趣学前端，可以私聊我，咱们探讨探讨。我对前端也学了些，怎么从小白起步我有经验。

  3. 有没有招 C / C++，同时要求懂一些前端？投这样的后端 + 前端，你的竞争力就提升了数倍！
