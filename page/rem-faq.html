找工作

[怎么找工作] 考研预计考不上，想找工作。没实习也没有时间实习，没方向，该怎么做才能找到工作。

更新：24-09-09 | 发布：24-09-09

[Q] 老师，我觉得自己考研很大可能考不上了，现在想找工作，我想问一下应该怎么找啊，我也没有实习，现在有课也没时间实习，没有方向。我应该怎么做啊

[S] 有以下建议，仅供参考。

考研能考上，也建议同步找工作。考研上岸了，所找到工作签订的合约自动解除的，不算违约。研究生毕业了，还是要找工作的。因此即使考研有把握，建议花点时间同步找工作，一是积累经验，二是有个 offer 在手就掌握主动。

升学就业，都很好。每年有那么多同学选择升学，同时也有那么多同学选择找工作赚钱啦，都很好。

把找工作当做“销售”，会很有意思。你代理自己，把自己“销售”给用人单位，需求是什么，自己如何满足，怎么争取好价格，……，都是有意思的事情。

先写个简历，说说自己有多厉害！这个学过，那个会做。上得厅堂，下得厨房。就是很厉害！如需要，我可协助优化简历。

积极投递，没啥损失。结合岗位要求 + 个人特长 + 兴趣爱好等，积极投简历。酒香不怕巷子深的年代已一去不复返，何况是找工作。不中也没啥损失。看看QQ群消息，以及智慧就业，各种学校的招聘会，就差不多了。

心态放平，选合适岗位。有时 100% 满足岗位要求，不等于拿到 offer。可能是岗位热门，应聘者人数远超过拟招人数，等等。

该出手时快出手。“招满为止”可能是公司招聘营销手段，也可能不是。相同或基本等同条件下，是会考虑先到先得的：确定下前两天面试通过的 A 同学是否接受 offer，如 OK 则今天投递的 B 同学就不面试了（因为条件看上去差不多，面试也是成本）。

勇敢投，不被招聘要求吓倒。招聘要求通常会描述很多，但真正的决定项（不满足就肯定不被录用）是较少的，其他不满足项通常只会减分。而且对于应届本科毕业生，很多企业更关注发展潜力，即：未来预计能成为年薪百万的人才，而此时只花费数十万就招募进公司了。

及时复盘，从小白到达人。要及时复盘每一次笔试、一面二面综面，发扬优点改进不足，不放过每个细节，全心全意做一次销售，把自己“卖”个好价钱！从找工作小白迅速成长为达人！

边投边学，打铁还需自身硬。遇到心仪岗位但自身能力有缺失，则要马上学习补齐。补齐方法要正确，教程要适合自己，在有限时间内补齐到能满足岗位要求即可。以投促学，以学辅投，交织进行，不可偏废。

秋招不行就春招。秋招能上岸是最理想的。通过秋招尝试，尽可能往前经历更多招聘环节，觉醒过来要加倍努力找工作，大致明白自己的最爱岗位是什么，为应聘其他企业或 25届春招（在 25年春季），也是特别有价值的。

4399-【2025校招】系统测试工程师
测试类 | 广州

[岗位描述]

1、负责账号中心、支付中心、活动推广等平台系统的测试工作；
2、负责项目管理、工作管理等企业系统的测试工作；
3、使用工具和脚本，提升测试效率，增加测试广度和深度；
4、梳理和推进流程，提升项目流畅度。

[岗位要求]

1、本科及以上学历，25届理工科专业，计算机、软件工程等相关专业优先；
2、掌握软件工程和软件测试理论，掌握常用测试设计方法（如边界值、等价类、因果分析）；
3、掌握一门编程或脚本语言（如C++/Java/Python/PHP/Shell），了解常用数据库（如Mysql）；
4、有互联网产品测试经验者优先，有测试工具（如Appium、JMeter）使用经验者优先；
5、有逆向思维，有责任心，有团队合作意识，有较强的推动及抗压能力。

[C/C++] 最理想最擅长 C 和 C++，其他的语言岗位有必要去试着投嘛

Q--2024/08/28

现在想法就是找开发岗，我最理想也最擅长肯定是 C 和 C++，其它的语言岗位我有必要去试着投嘛，因为我感觉现在学可能多少有点赶。我投过前端开发的，我其实这个方面也感兴趣，但毕竟接触的少，投了就没啥后续了

S--2024/08/29

以下是建议，仅供参考。预祝同学早日找到心仪岗位！

  1. 聚焦投 C 和 C++ 岗位
    • 擅长 + 理想，因此建议聚焦。
    • C / C++ 岗位的薪酬也不错，如能录用，还喜欢工作内容，就很理想！
    • 专业技能（C / C++），以及软能力（沟通 / 表达 / 逻辑 / ……），都符合要求，才能拿到 offer。各方面都不能偏废。
    • 即使满足岗位诸方面要求，也不一定能拿到 offer。可能有更厉害的同学也投递了，尤其是其他 985 和 211 学校的。因此，要能抗压，要尽早动手，则机会更多！
    • 即使很顺利很厉害，手握 10 个 offer，最后也只能去一个单位。因此，心态要平！这么多岗位，我只要找 1 个，肯定是能找到的。
    • 瞄准可能会被录用的企业的岗位。尽量不要在太热门的太高薪的岗位上花太多时间，都知道好，很多牛娃也投，所以和咱们普通人就关系不大。但可能情况下还是要踮脚跳一跳，力争找一个自己能力所匹配薪酬的 1.2 倍甚至 1.5 倍的岗位。
    • 多投、多总结复盘。看看还有哪些不足的，赶紧补上。补的目标是通过面试找到工作，不要无限制的补，没那么多时间。原先只能找 10000 元岗位的，通过不断投岗位、不断获取面试找工作经验、不断补齐能力，最终可能找到 15000 甚至 20000 元的岗位。

  2. 不放弃前端的学习
    • 24 届校友L，通过在 23 年下半年自学增强 Java 技能，花费2~3个月，最终入职了国内著名人工智能上市公司，做 Java 开发。想说的是：如果要学，是来得及的。
    • 学到什么程度？能通过面试找到工作就行。不要无限制的学习，没有那么多时间。
    • 如果有兴趣学前端，可以私聊我，咱们探讨探讨。我对前端也学了些，怎么从小白起步我有经验。

  3. 有没有招 C / C++，同时要求懂一些前端？投这样的后端 + 前端，你的竞争力就提升了数倍！
